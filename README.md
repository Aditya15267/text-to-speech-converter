# Text-to-Speech Converter

A simple Python-based Text-to-Speech (TTS) converter using the `pyttsx3` library. This project allows user to input text and hear it spoken aloud by the system's text-to-speech engine.

## Features

- Converts text into speech.
- Works offline
- Lightweight and easy to use
- Support different voices and speech rates

## Installation

Make sure you have Python installed on your system. Then, install the required library:
    
```sh
pip install pyttsx3            
```

## How to use

1. Clone the repository:
    ```sh
    git clone https://github.com/Aditya15267/text-to-speech-converter.git
2. Run the script:
    ```sh

3. Enter any text when prompted, and the system will read it aload.
